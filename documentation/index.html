insert-nth

insert-nth($list, $index, $value)
View source

Parameters

Argument Type Default Role
$list List - list
$index Number - index to add to
$value List - value to add

Return

  • List | false

Raise

  • [Error] $index must be a number
  • [Error] $index must be a non-zero integer

Examples

insert-nth(a b c, 2, z) 
// a z b c

// false (error)

// false (error)
insert-nth(a b c, 10, z) 
// a b c z

slice

slice($list, $start: 1, $end: length($list))
View source

Slices $list between $start and $end.

Parameters

Argument Type Default Role
$list List - list
$start Number 1 start index
$end Number length($list) end index

Return

  • List | false

Raise

  • [Error] $start and $end must be numbers
  • [Error] $start and $end must be non-zero positive integers
  • [Error] $start must be lesser or equals to $end
  • [Error] $start and $end must be lesser than or equals to length($list)

Examples

slice(a b c d,  2, 3)
// b c
slice(a b c d,  3, 2)
// false (error)
slice(a b c d,  3, 5)
// false (error)
slice(a b c d, -1, 3)
// false (error)
slice(a b c d,  0, 3)
// false (error)
slice(a b c d,  3, 3)
// c
