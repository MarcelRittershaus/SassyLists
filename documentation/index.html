    Slices $list between $start and $end.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $list

    list to slice

    		 List — none
    $start

    start index

    		 Number 1
    $end

    end index

    		 Number length($list)

    Returns

    List, Null

    Example

// b c d

    • End index has to be lesser than or equal to list length for sl-slice.
